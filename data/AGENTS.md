# data/ - Data Files & Targets

**Purpose:** Static data files and test database artifacts.

## FILES

| File | Purpose |
|------|---------|
| `targets.txt` | Target list for blast operations (phone numbers, usernames) |
| `test.db-shm` | SQLite WAL shared memory (test DB artifact) |
| `test.db-wal` | SQLite WAL file (test DB artifact) |

## NOTES

- `targets.txt` is read by `src/utils/randomTargets.ts` for blast targeting
- Test DB files are generated by `npm run db:init` in test environments
- `.gitkeep` preserves directory in git; actual DB files may be gitignored
