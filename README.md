# Joki Blast Engine

Welcome! This is the Joki Blast Engine repository. This page provides a simple entry point to navigate the project and get started quickly.

---

## Quick Start

To launch the project, use the following commands:

- On Linux/MacOS:
```bash
./start.sh
```

- On Windows:
```powershell
.\start.ps1
```
