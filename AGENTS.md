# joki-blast-engine

## Purpose
Multi-platform blast engine for high-throughput message distribution. Supports WhatsApp, Telegram, Instagram, and Twitter adapters. Built with Node.js/Express and Next.js dashboard.

## Key Files
| File | Description |
|------|-------------|
| `package.json` | Dependencies and scripts |

## Subdirectories
| Directory | Purpose |
